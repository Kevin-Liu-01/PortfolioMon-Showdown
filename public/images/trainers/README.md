# Generation V trainer sprites

PortfolioMon includes the complete 258-file Generation V trainer-sprite category from Bulbagarden Archives, plus the available matching overworld sprites used by chibi mode.

- 258 battle sprites (246 front-facing and 12 back-facing)
- 254 trainers with an overworld/chibi match
- 142 unique chibi image files (some poses are shared by related battle entries)
- 4 full-size-only fallbacks where no matching overworld sprite is available

The roster records the source filename and source page for every selection in `src/data/genVTrainerRoster.json`. Re-download and verify the local asset set with:

```sh
npm run sync:trainers
```

Sources:

- <https://archives.bulbagarden.net/wiki/Category:Generation_V_Trainer_sprites>
- <https://archives.bulbagarden.net/wiki/Category:Overworld_Trainer_sprites>
